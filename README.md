### Development

Basta digitar no terminal na pasta do projeto:

```sh
$ npm run start
```

Ou se estiver usando yarn:

```sh
$ yarn start
```

#### Building for source

Basta digitar no terminal na pasta do projeto:

```sh
$ npm run build
```

Ou se estiver usando yarn:

```sh
$ yarn build
```
